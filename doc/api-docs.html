messages

Message operations

Send message

Sends a new message to either a group conversation or private conversation. If conversation doesn't exist between users and no conversationId is provided, creates a new private conversation.

Authorizations:
bearerAuth
Request Body schema: multipart/form-data
required
senderUsername
required
string

Username of the sender

receiverUsername
string

Username of the receiver (required for private conversations, optional for groups)

conversationId
string

Conversation ID (optional, used for group messages or existing conversations)

content
required
string

Message content

attachments
Array of strings <binary> [ items <binary > ]

Optional file attachments

Responses

Test server running locally, for development purposes.

http://localhost:3010/messages

Response samples

  • 201
Content type
application/json
{
  • "id": "54321",
  • "senderId": "Karol",
  • "content": "Hello, whats up?",
  • "conversationId": "12345",
  • "timestamp": "2026-01-01T21:37:00Z"
}
